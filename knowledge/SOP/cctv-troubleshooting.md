# CCTV/NVR troubleshooting

ใช้กับเคสกล้องดูไม่ได้ ภาพดำ ภาพค้าง ขึ้น offline หรือ no signal

ควรถามข้อมูลหลัก:
- ชื่อกล้องหรือบริเวณ เช่น CAM-WH-01 หน้าโกดัง
- ดูไม่ได้ทุกเครื่องหรือเฉพาะเครื่องผู้ใช้
- อาการ: ภาพดำ, no signal, offline, ภาพค้าง, playback ไม่ได้
- เริ่มเกิดเมื่อไหร่ และมีไฟดับ/ซ่อมบำรุง/เปลี่ยน network ล่าสุดไหม
- มี screenshot หรือ error จาก NVR/app ไหม

ทีมเกี่ยวข้องเมื่อจำเป็น: Network, Vendor, Security
