# Printer troubleshooting

ใช้กับเคสเครื่องพิมพ์/ปริ้นเตอร์พิมพ์ไม่ได้ กระดาษติด offline print queue ค้าง หรือขึ้น error

ควรถามข้อมูลหลัก:
- ชื่อเครื่องพิมพ์หรือจุดติดตั้ง เช่น ห้องบัญชี, HR, หน้าโกดัง
- อาการที่พบ: offline, print queue ค้าง, กระดาษติด, หมึกหมด, error code, พิมพ์ช้า, พิมพ์ออกมาไม่ครบ
- เป็นทุกเครื่องหรือเฉพาะเครื่องผู้ใช้คนเดียว
- ผู้ใช้อื่นพิมพ์เครื่องเดียวกันได้ไหม
- มีเครื่องพิมพ์ตัวอื่นใช้แทนได้ไหม

ทีมรับผิดชอบเริ่มต้น: IT Support
ทีมเกี่ยวข้องเมื่อจำเป็น: Vendor ถ้าเป็น hardware/service contract
